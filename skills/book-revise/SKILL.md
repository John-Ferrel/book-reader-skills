# Book Revise

Use this skill after a reviewer has produced
`review/revision-plan-round-<n>.md`.

Responsibilities:

- read review findings and revision plan
- re-read relevant source blocks
- apply revisions to notes, threads, model, indexes, guide, and workspace state
- preserve or mark superseded prior claims
- remove/collapse/defer redundant artifacts when review asks and evidence
  supports the action
- write `revisions/revision-log-round-<n>.md`
- update `workspace.json` to `workspace_stage: revised`,
  `revision_status: applied`, and `current_required_action: run-verify`

Reviewer and reviser are separate roles.

Read `references/revision-protocol.md`.
