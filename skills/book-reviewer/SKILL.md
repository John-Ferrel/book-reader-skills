# Book Reviewer

Use this skill to audit inference discipline in an existing workspace.

Reviewer writes round-specific files:

- `review/review-report-round-<n>.md`
- `review/revision-plan-round-<n>.md`

It may also update latest copies `review/review-report.md` and
`review/revision-plan.md`. Do not modify non-review files unless the user
explicitly requests repair mode.

Read:

- `references/inference-discipline-audit.md`
- `references/reviewer-boundary.md`

Reviewer must include artifact redundancy audit, anti-laziness audit,
traceability audit, lens compliance audit, and state consistency audit.
