# Book Reconstruct

Use this skill for the core evidence-driven reading pass after intake.

Read these references:

- `references/evidence-discipline.md`
- `references/reconstruction-model.md`
- `references/knowledge-organization.md`
- `references/continuation-planning.md`
- `references/anti-laziness-obligations.md`

Workflow:

1. Inspect source metadata, limitations, source-map, and normalized source.
2. Confirm, merge, split, or discard unit candidates into semantic units.
3. Write note items: observation, paraphrase, inference, uncertainty, question,
   and external-context if needed.
4. Build integration threads.
5. Build reconstruction model files.
6. Build useful indexes or explicitly defer/block/mark not-applicable.
7. Build guide and continuation plan.
8. Self-audit traceability before reporting state.

Do not claim completion without independent review. If reconstruction is done
but review has not run, report `review-required`.
